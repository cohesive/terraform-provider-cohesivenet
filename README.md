# Terraform Provider for Cohesive Networks VNSCubed & MS

Crude steps to get up and running: 
Install Go - $brew install go

1. Git Clone
2. go mod vendor
3. make install
4. cd examples
5. Update main.tf with username, password, api token
6. Terraform init
7. Terraform apply
